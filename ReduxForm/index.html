Redux Form v8.3.0

Step #1

Get ready with npm-create-react app

Step #2

Create a new directory 'store' in src folder

            import { createStore, combineReducers } from 'redux'
            import { reducer as formReducer } from 'redux-form'

            const rootReducer = combineReducers({
                // ...your other reducers here
                form: formReducer
            })
            const store = createStore(rootReducer)

            export default store;
Step #3

Import provider from redux-form

            import { Provider } from 'redux-form';

Import this store/index.jsx in App.js

And wrap <App /> with <Provider> tag and pass store as value

             <Provider store={store}>
                 <App />
             </Provider>,
            document.getElementById('root')
Step #4

In <App/> import reduxForm, fields

            import { reduxForm, fields} from 'redux-form';
